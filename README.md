# Where Should I Go Eat?
If you're hungry and unsure of where to go, this app randomly tells you where to go eat with a click of a button.
## About The Project
I wrote this app about two and half years ago when I was working at a particular job that paid for the team's lunch on Fridays. We, the team, could never come to a majority decision so I wrote this program to give us a random choice.

Yeah, we didn't stick to the choice either. :smiley:
